◆ FindApriltags()

int OpenCVwrapper::FindApriltags ( cv::Mat  InputImage)

Odnajdywanie tagów na podstawie pobranego zdjecia Ze zdjęcia pozyskiwana jest wysokość i szerokość, następnie algorytm detekcji szuka na zdjęciu AprilTagów.

Jeśli tag zostanie znaleziony, zwracane jest jego ID. Jeśli na zdjęciu znajduje sie więcej niż jeden tag (do czego nie powinno dojść), wtedy aplikacja zwraca tag o mniejszym ID.

Parametry
InputImageto matryca posiadajaca informacje na temat obrazu przesłanego z aplikacji Android
Zwraca
Zwraca informacje - jaki tag został znaleziony przez system
