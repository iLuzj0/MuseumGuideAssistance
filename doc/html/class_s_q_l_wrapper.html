nlohmann::json SQLWrapper::SQLGetExibitsIntoJSON ( std::string  tagID)

Metoda, która na podstawie ID znalezionego tagu zwraca nam informacje o eksponacie, który stoi za danym ID.

Parametry
tagIDZmienna typu string, która przekazuje do bazy danych w zapytaniu ID taga. Na podstawie ID ustalane są informacje na temat eksponatów
Zwraca
Zwraca zmienną typu JSON, która zawiera w sobie informacje na temat eksponatu - nazwe autora, nazwe eksponatu, opis eksponatu, date utworzenia
